# Report on Firewalls, IDS, and IPS

## Introduction

Network security is an important part of cybersecurity. Organizations use security technologies such as firewalls, Intrusion Detection Systems (IDS), and Intrusion Prevention Systems (IPS) to protect their networks and data from threats.

## Firewalls

A firewall acts as a barrier between trusted and untrusted networks. It examines incoming and outgoing traffic and allows or blocks communication based on security rules.

### Types of Firewalls

1. Packet Filtering Firewall – Filters packets based on IP addresses, ports, and protocols.
2. Stateful Inspection Firewall – Tracks active connections and makes decisions based on connection state.
3. Proxy Firewall – Acts as an intermediary between users and internet resources.
4. Next-Generation Firewall (NGFW) – Provides advanced security features such as application control and threat detection.

## Intrusion Detection System (IDS)

An IDS continuously monitors network traffic and system activities for suspicious behavior. When a potential threat is detected, the system generates alerts for administrators.

## Intrusion Prevention System (IPS)

An IPS performs the same monitoring functions as an IDS but also takes action to stop threats automatically by blocking malicious traffic.
